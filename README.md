# TTY Garden Configuration

This repo hosts the core nixos configuration that operates tty.garden and its
subdomains.

## Current features:
- Automatic SSL certificates via ACME DNS challenges
- Automatic backups with restic
- NGINX webserver
- ATProto PDS at https://pds.tty.garden
- Gitea instance at https://seed.tty.garden
- Software mirrors at https://mirror.tty.garden
